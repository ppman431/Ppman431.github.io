Advanced Boat Controller System

Vehicles ¬∑ Physics

A physics-based boat controller with smooth steering, speed curves, and buoyancy. Built for ocean / naval games that need responsive, controllable boats instead of default Roblox behavior.

  • Uses BodyGyro + VectorForce for stable orientation & movement
  • Configurable BoatStats module for acceleration, max speed & turning
  • Server-authoritative movement with client smoothing
Boat.BodyGyro.CFrame = CFrame.new(Boat.Position, Boat.Position + direction)
Boat.Velocity = Boat.Velocity:Lerp(direction * speed, 0.1)

Minimap With Fog-of-War

UI ¬∑ Exploration

A minimap that reveals only explored areas, perfect for exploration or survival games. Uses UI masks to smoothly uncover regions as the player moves.

  • Chunk-based reveal system for performance
  • Fog-of-war using mask circles that expand over time
  • Local-only updates to reduce server load
revealCircle.Size += UDim2.fromOffset(1, 1)
maskGroup:AddChild(revealCircle)

Flintlock / Weapon System

Combat ¬∑ Raycasting

A custom weapon system using raycasts for hitscan bullets, with reloads, bloom, hit-effects, and server-side validation to prevent exploits.

  • Raycast-based bullet firing with filter params
  • Server-side damage handling & anti-exploit checks
  • Supports reload animations, sounds, and UI cooldowns
local result = workspace:Raycast(origin, direction * range, params)
if result then
    Remotes.Damage:FireServer(result.Instance, damage)
end

Diving System + Air Bar

Underwater ¬∑ UI

An underwater mechanic that tracks air, applies drowning damage, and animates an on-screen air bar. Automatically detects when the player is in water.

  • Air drain and recovery with adjustable difficulty
  • Drowning damage when air hits zero
  • Smooth UI bar using TweenService or direct resizing
air = math.clamp(air - drainRate * dt, 0, maxAir)
AirBar.Size = UDim2.new(air / maxAir, 0, 1, 0)
